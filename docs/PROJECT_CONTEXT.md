# ATLAS PROJECT CONTEXT

**Project Name:** Atlas Visual Engine

## Vision

Atlas Visual Engine is a production-quality Python library for creating educational diagrams, animations, experiments, and visual assets for Physics, Chemistry, and Mathematics.

The project is designed to generate consistent, reusable, publication-quality educational graphics.
